# two filters equivalent to one

    Code
      df1 %>% remote_query()
    Output
      <SQL> SELECT *
      FROM `df`
      WHERE (`x` > 3.0) AND (`y` < 3.0)

---

    Code
      df1 %>% remote_query()
    Output
      <SQL> SELECT `x`, `y`
      FROM (
        SELECT `x`, `y`, AVG(`x`) OVER () AS `q01`
        FROM `df`
      )
      WHERE (`q01` > 3.0) AND (`y` < 3.0)
